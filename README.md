# Reverberation
ノベルゲーム「夜空の贈り物」のソースコードと完成品、windows vista/7/8/8.1/10 対応 (のはず)  

[公式サイト](https://kawashi.github.io/ReverberationSites/)  

# フォルダ構成
- source - .cpp ファイルが入ってる
- header - .h ファイルが入ってる
- resource - 圧縮した音楽や画像が入ってる
- 夜空の贈り物ver1.01 - コンパイルして生成された.exeが入ってる
